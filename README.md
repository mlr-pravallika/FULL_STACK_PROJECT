# InteriorDesign — Product Showcase

Static portfolio & product showcase for an interior-design studio.

Features:
- Responsive gallery with category filters
- Modal product details & enquiry CTA
- Accessible HTML & semantic structure
- Deploy-ready static site (GitHub Pages)

How to run locally:
1. Open `index.html` in browser (or use a static server)
2. Replace images in `assets/images` with your product photos
3. Edit copy, prices, and email in the markup
